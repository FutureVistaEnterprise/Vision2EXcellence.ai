# Vision to Excellence

**Operations Excellence Framework** — Transforming organizations through strategic alignment, operational maturity, and a relentless pursuit of world-class performance.

## Overview

Vision to Excellence (V2E) is a holistic operations-excellence framework designed to bridge the gap between strategic ambition and operational reality. It provides organizations with a structured, repeatable pathway from initial maturity assessment through continuous, measurable improvement.

## Website Structure

This is a single-page website built with pure HTML, CSS, and JavaScript — no frameworks, no build tools, no external dependencies.

### Sections

- **Hero** — Brand introduction and call-to-action
- **About** — Philosophy and framework overview with key stats
- **Maturity Journey** — Six interactive pillar cards (Start Your Journey, Maturity Assessment, Transformation, Professional Excellence, Performance Management, Project Readiness)
- **Excellence Framework** — Strategic-to-operational layered architecture (Strategic, Tactical, Enablement, Operational)
- **Service Disciplines** — Eight service cards covering Continuous Improvement, Document Management, IMS, Demand Forecasting, Assets and Facilities, Lean Six Sigma, Portfolio Management, Program Management
- **Agentic AI** — AI workflow automation section with agent pipeline visualization
- **Contact** — Email contact info and intake form collecting visitor information

### Features

- Fully responsive (desktop, tablet, mobile)
- Scroll-reveal animations via IntersectionObserver
- Interactive modal popups for each service and journey pillar
- Sticky navigation with scroll-aware styling
- Contact form with validation and localStorage persistence
- Zero external dependencies — works offline

## Deployment

This site is designed for **Cloudflare Pages** deployment:

1. Push this repository to GitHub
2. Connect the repository in Cloudflare Pages
3. Set build output directory to `/`
4. Deploy

The site will be live at `your-project.pages.dev`. Add a custom domain in Cloudflare Pages settings.
